# وضع المحاكاة (Simulation Mode)

## نظرة عامة

وضع المحاكاة يسمح بتشغيل النظام بدون الحاجة إلى قاعدة بيانات SQLite. يتم استخدام بيانات وهمية (mock data) مخزنة في الذاكرة.

## كيفية التشغيل

### Windows:
```bash
.\bin\RestaurantManagementSystem.exe --sim
```

### Linux/macOS:
```bash
./bin/RestaurantManagementSystem --sim
```

## المميزات

- ✅ لا حاجة لقاعدة بيانات
- ✅ بيانات تجريبية جاهزة
- ✅ مثالي للاختبار والتطوير
- ✅ سريع في التشغيل

## ملاحظات مهمة

⚠️ **تحذير**: في وضع المحاكاة:
- البيانات لا تُحفظ بعد إغلاق التطبيق
- جميع التعديلات مؤقتة
- لا يمكن استرجاع البيانات بعد الإغلاق

## الاستخدام

1. شغّل التطبيق مع `--sim`
2. سجل الدخول بأي من المستخدمين التجريبيين
3. استخدم النظام بشكل طبيعي
4. جميع البيانات ستكون في الذاكرة فقط
